Real-Time Delivery Monitoring and Tracking System

This is a web-based application for real-time monitoring delivery staff in various ways, including determining their delivery schedule, current location, annotations, work efficiency, report generation etc. It was created using Laravel as a framework and HTML, CSS, JavaScript, PHP, AngularJS and AJAX as languages. Google maps were used with several customized features. It was a team project. The delivery staff and managers of DataNext Corporation (Okinawa, Japan) are using this application and it is working pretty good.
